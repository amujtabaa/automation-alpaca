# Generic Chat Adapter

When no repository-integrated agent runtime exists:

1. Paste `templates/fable-core-v3.md`.
2. Paste the work order.
3. Paste only the PKL pages named by the work order.
4. Use DELEGATED verification mode if the model cannot run commands.
5. Require DONE evidence before accepting status.

Do not paste the entire operating system into the chat.
