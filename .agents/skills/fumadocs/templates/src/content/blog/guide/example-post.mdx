This is an example post demonstrating the frontmatter contract.

- `menu` is what appears in the SIDEBAR (short).
- `title` is the H1 on the page AND the SEO title in search results.
- `seoTitle` overrides the search title ONLY when `title` exceeds 60 characters.
- `description` becomes the meta description with the brand prefix auto-prepended.
- `views` seeds popularity sorting (the "Popular" toggle on the blog landing page).

## Writing posts

Standard MDX. Headings build the table of contents automatically.

After creating a post: register its slug in `blog-structure.ts`, then run `pnpm generate:meta`.
